1、VS项目源码以及可执行程序J_ins.exe均在Jarvis_Inspection.7z中，解压到文件夹后，可以发现其中含有漏洞检测样本库以及同源性检测样本库两个压缩文件，请直接解压到项目所在文件目录中，并请按照A样本库使用说明进行检查。
2、报告为软件安全课程设计-1801-杨傲.doc，可直接打开。
3、录屏压缩文件为1801-杨傲-演示录屏-10-9.7z，解压后为视频文件可播放。
4、内部的qt半成品仅用来表示确实尽力尝试了qt的使用，碍于时间关系转向了命令行。
5、此处两个样本库和项目目录下两个压缩包中内容是一样的，可以不看。由于任务书中我选择了两个样本库的任务，分别是：
（1）同源性样本库50个样本(每个100行以上)每10个为相似的一组，1分，最高5分
（2）漏洞检测与同源性检测样本库，样本数量不少于10个，每个代码行数不少于100行；每种漏洞至少一个。
所以在第二个任务中不再写同源性检测的样本毕竟第一个任务中已经有了五十个，而是写了十个漏洞检测样本。
样本库使用说明已在项目目录下，解压即可看到。
